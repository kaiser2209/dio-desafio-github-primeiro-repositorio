Comandos Git

git clone {repositório github}	(Clonar para o repistório local)
git status			(Status do Repositório)
git add .			(Adicionar todos os arquivos)
git commit -m "mensagem"	(Faz o commit das alterações com mensagem definida pelo usuário)
git push origin main		(Envia o commit para o github)
